Why RNG Games?

  • RNG Games is the first project during my bootcamp at Ironhack, a well known tech school.

    • What does RNG mean?

  • A Random Number Generator (RNG) is a mathematical construct, either computational or as a hardware device that is designed to generate a random set of numbers that should not display any distinguishable patterns in their appearance or generation, hence the word random.

    • I don't see how that relates to gaming...

  • In gaming it essentially means there is some level of random luck involved in the game.

    • So...What does RNG Games do?

  • The website uses RAWG free API to fetch and display some data and pictures from random games. Maybe you can find some game you would never hear about if it wasn't for my website.
